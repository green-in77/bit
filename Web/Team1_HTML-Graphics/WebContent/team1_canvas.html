canvas : 그림이 그려질 수 있는 빈 박스

JavaScript를 통해 박스안에 그림(경로, 상자, 원, 텍스트,이미지 등)을 그릴 수 있다.

convas는 그냥 빈 박스다. 빈 박스안에 JavaScript를 사용해서 그린다.

너의 브라우저는 HTML5를 지원하지 않아!!!!!!!!!!
canvas id="컨버스이름" width가로(픽셀) / height세로(픽셀)
strokeStyle="색상명" : 선 색상
strokText : 글씨의 외곽선을 그린다.
너의 브라우저는 HTML5를 지원하지 않아!!!!!!!!!!
getContext : 컨버스에 그릴 수 있는 속성을 가져옴
moveTo : 포커스이동 / stroke() : 출력
beginPath : Path경로시작 / closePath() : Path 시작점으로 이동
lineTo:선을 그려준다
arc(x,y,반지름,시작값,끝값,시계방향) : 원을 그려준다
너의 브라우저는 HTML5를 지원하지 않아!!!!!!!!!!
createLinearGradient(시작점x1,시작점y1,끝점x2,끝점y2) : 선형
createRadialGradient(x1,y1,1번 반지름,x2,y2,2번 반지름) : 원형
addColorStop : 그라데이션 색 설정(0~1 사이 그라데이션)
fillStyle : 채우기 / strokeStyle : 선 색상 /lineWidth : 선 두께
strokeRect / fillRect(여백x, 여백y, W, H) : 테두리만 / 채워진 도형
roundedRect(시작점 x, 시작점 y, 사각형 w, 사각형 h, 곡선의정도)
: 모서리가 둥근 사각형
fillStyle : 채우기
fillRect (시작점 x, 시작점 y, 사각형 x, 사각형 y) : 사각형
fill(): 도형의 안쪽을 채워넣는다.
stroke(): 안을 비우고 도형의 선만 그린다.
bezierCurveTo(기준x1, 기준y1, 기준x2, 기준y2, 종료x3 , 종료y3)
: 곡선을 그리게 해주는 함수.
var img = new Image():이미지 객체를 생성
img.onload = function(){} : 이미지 로딩이 완료되면 실행
drawImage(img객체,시작점 x, 시작점 y, 이미지 w ,이미지 h)
img.src="이미지 경로" : img 객체에 들어갈 이미지소스의 경로
