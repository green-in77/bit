SVG : 백터(vector) 이미지를 표현하기 위한 포맷으로 w3c에서 만든 백터 이미지 표준입니다.

SVG 코드 설명 : 1. SVG 이미지는 svg 요소로 시작합니다 2.svg 요소의 너비와 높이 속성은 SVG 이미지의 너비와 높이를 정의합니다

1.circle 요소는 원을 그리는 데 사용됩니다
2. cx 및 cy 속성은 원 중심의 x 및 y 좌표를 정의합니다.
3. r 속성은 원의 반지름을 정의합니다
4. 획 및 획 너비 속성은 모양의 윤곽이 나타나는 방식을 제어합니다.
5. 채우기(fill) 속성은 원 안의 색상은 노란색으로 설정했습니다
1. rect 요소의 (width:400 height:110) 사각형의 높이와 너비를 정의함.
2. style 속성은 사각형의 CSS 속성을 정의하는 데 사용.
3. fill 사각형의 채우기 색상을 정의함. (fill:rgb(0,0,255))
4. 사각형 테두리(stroke-width)의 너비를 4로 지정.
5. 사각형 테두리의 색상(stroke)을 정의합니다.
1. x,y속성은 사각형의 왼쪽 및 상단의 위치를 정의
2. 사각형 테두리(stroke-width)의 너비를 5로 지정.
3. 획 색상의 자체 투명도(opacity )를 0.5 지정
4.viewBox : 커버 할 svg의 사각형 영역을 정의합니다.
5. preserveAspectRatio : 뷰포트가 변경되면 스트레치하는 방법을 정의.
1. 라임색 채우기
2. 선색 : 보라색
3. 테두리(stroke-width)의 너비를 5로 지정
4. fill-rule : 채우기 규칙 속성을 "evenodd"로 지정.
1. linearGradient 요소는 선형 그래디언트를 정의
2. linearGradient 요소는 defs 태그 내에 중첩되어야함
3. 그라디언트의 색상 범위는 둘 이상의 색상으로 구성 될 수 있다
4. 각 색상은 stop 태그로 지정됩니다
5. 오프셋 속성은 그라디언트 색상이 시작되고 끝나는 위치를 정의
1. 획 속성은 요소의 선, 텍스트 또는 윤곽선의 색상
2. g 태그 안에 path태그를 넣어야한다.
3. 선에도 그라디언트 속성을 넣을 수 있다.
