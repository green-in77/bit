요즘 배우고 있는 것들

JAVA, JavaScript, SQL 등등 웹 개발을 위해 열심히 공부하고 있습니다.

JAVA

한국에서는 정부나 기업의 시스템 통합 프로젝트가 대부분 자바로 구현되기 때문에 자바는 기업용 시장에서 두각을 나타내고 있습니다.

JavaScript

JavaScript는 웹페이지를 동적으로, 프로그래밍적으로 제어하기 위해서 고안된 언어입니다. 그렇기 때문에 오늘날 가장 중요한 플랫폼이라고 할 수 있는 웹브라우저에서 유일하게 사용할 수 있는 프로그래밍 언어입니다.

SQL

SQL은 Structured Query Language의 약자로서 구조적 질의 언어입니다. SQL은 데이타베이스의 데이타를 쿼리하거나 처리하는데 사용되고 있습니다. 모바일 버전인 SQLite도 있습니다.
