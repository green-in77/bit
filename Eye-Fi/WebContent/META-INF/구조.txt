					- common : include 되는 page
						- admin.jsp : 관리자가 아니면 화면에 접근 불가
						- member.jsp : 회원이 아니면 화면에 접근 불가
						- top.jsp : 상단
						- bottom.jsp : 바닥
						
					- lib : 라이브러리
						JSTL, DB, Json, smtp
						
					- views : UI 페이지
						- 
					
					- process : 처리하는 jsp
						- redirect.jsp : 화면이동처리
						- UsingConnection.jsp : DB연결 확인페이지
